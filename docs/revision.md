# 동물 월드컵 - 변경 이력

## 개요
이 문서는 "동물 월드컵" 프로젝트의 주요 변경 사항을 시간 순으로 정리한 것입니다.

---

## 2025-11-02

### Phase 1: 프로젝트 컨셉 변경
**변경 내용**: Unsplash 기반 이미지 월드컵 → 동물 테마 월드컵으로 전환

**주요 작업**:
- Unsplash 관련 코드 전면 제거
- 무료 동물 이미지 API 통합 (Dog CEO API, The Cat API, RandomFox API)
- 카테고리 구성: 강아지, 고양이, 여우, 믹스 (4개)
- 타이틀 변경: "동물 월드컵"
- 토너먼트 규모: 32강 → 16강으로 조정

**이유**: API Key 없이 사용 가능한 무료 API 활용, 동물 테마로 명확한 타깃팅

---

### Phase 2: UI/UX 개선 (1차)
**변경 내용**: 로딩 화면, 버튼, 카테고리 카드 UI 개선

**주요 작업**:
- 로딩 화면 정렬 수정 (flex-direction: column)
- 스피너 내부 이미지 회전 애니메이션 추가
- 버튼 크기 확대 (sm→md, padding 증가)
- 카테고리 카드에 실제 동물 이미지 표시
- 랜덤 선택 문구 100개 추가 ("Pick me!", "날 선택해!" 등)
- 믹스 카테고리 이미지 로딩 재시도 로직 강화 (5회)

**이유**: 사용자 피드백 반영, 가독성 및 인터랙션 개선

---

### Phase 3: 카테고리 조정 및 스피너 개선
**변경 내용**: 새(Birds) 카테고리 제거, 믹스 비율 조정, 스피너 애니메이션 수정

**주요 작업**:
- Birds 카테고리 제거 (API 응답 불안정)
- 믹스 카테고리 비율 조정: 강아지 5장, 고양이 6장, 여우 5장 (총 16장)
- 스피너 이미지가 박스와 함께 회전하도록 애니메이션 수정
- 결과 화면 텍스트 변경: "Photo from Picsum" → "I'm the Best!" → "Best Of Best!"

**이유**: 안정성 향상, 일관된 사용자 경험 제공

---

### Phase 4: 카테고리 카드 하이라이트 개선
**변경 내용**: 선택된 카테고리 시각적 강조, Mix 카테고리 고정 이미지 추가

**주요 작업**:
- 선택된 카테고리 하이라이트 강화
  - 배경색: primary color (핑크)
  - 테두리: accent color (노란색) 4px
  - 그림자: 8px 노란색 픽셀 그림자
  - Transform: translate(-4px, -4px)
- Mix 카테고리에 Unsplash 고정 이미지 추가
- Footer 추가: Copyright © 2025 by KwonYoungHu @Kookmin University

**이유**: 사용자가 선택한 카테고리를 명확하게 인식할 수 있도록 개선

---

### Phase 5: 카테고리 이미지 캐싱 및 텍스트 강조
**변경 내용**: 이미지 리프레시 방지, 텍스트 가독성 개선, 전체 폰트 크기 확대

**주요 작업**:
- **이미지 캐싱 시스템 구현**
  - `categoryImageCache` 객체 추가
  - 카테고리 선택 시 이미지 재로딩 방지
- **카테고리 텍스트 박스 처리**
  - 반투명 검은 배경 (rgba(0, 0, 0, 0.85))
  - 테두리 및 그림자 추가
- **전체 폰트 크기 1.3배 확대**
  - xs: 10px → 13px
  - sm: 12px → 16px
  - md: 14px → 18px
  - lg: 18px → 23px
  - xl: 24px → 31px
  - 2xl: 32px → 42px
- **UI 레이아웃 조정**
  - 카테고리 그리드: minmax(220px), max-width 900px
  - 카테고리 카드: min-height 240px
  - 랭킹 그리드: minmax(120px)

**이유**: 사용자 피드백 반영, 가독성 대폭 향상

---

### Phase 6: 카테고리 카드 레이아웃 전면 개편
**변경 내용**: 이미지 오버레이 → 이미지 + 텍스트 분리 레이아웃, 4개 카테고리 1줄 배치

**주요 작업**:
- **레이아웃 변경**
  - 이전: 텍스트가 이미지 위에 오버레이
  - 이후: 이미지 영역(상단) + 텍스트 영역(하단) 분리
- **그리드 조정**
  - `grid-template-columns: repeat(4, 1fr)` → 4개 카테고리 한 줄 배치
  - max-width: 1000px
- **Mix 카테고리 변경**
  - Unsplash 고정 이미지 제거
  - 큰 '?' 텍스트로 대체 (120px, 노란색, 픽셀 그림자)
- **반응형 레이아웃**
  - Desktop (>1024px): 4개 컬럼
  - Tablet (768-1024px): 2개 컬럼
  - Mobile (<768px): 1개 컬럼

**이유**: UI 일관성 개선, 텍스트 가독성 향상, 4개 카테고리 동등한 시각적 비중

---

## 기술적 개선 사항

### 이미지 로딩 시스템
- Fisher-Yates 셔플 알고리즘으로 랜덤 순서 보장
- 재시도 로직 (최대 5회)
- API 요청 간 100ms 딜레이 (레이트 리밋 방지)
- 진행률 표시 및 실시간 업데이트

### 스타일 시스템
- CSS Variables 활용한 테마 시스템
- 픽셀 아트 감성의 일관된 디자인 언어
- 반응형 그리드 시스템
- 애니메이션 및 트랜지션 효과

### 상태 관리
- localStorage 기반 세션 저장
- 카테고리 이미지 캐싱
- 플레이 히스토리 관리

- `scripts/main.js`: 동물 API 통합, 이미지 캐싱, 카테고리 카드 구조 변경

- `styles/variables.css`: 폰트 크기 1.3배 확대

- `styles/pixel.css`: 버튼 크기 확대

### HTML
- `index.html`: Press Start 2P 폰트 추가, Unsplash 스크립트 제거
